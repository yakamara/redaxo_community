
> Hinweis: Sollen keine "Terms of Use" abgefragt werden, dann diese Zeile:
```
checkbox|termofuse_accepted|Ich habe die Nutzungsbedingungen akzeptiert.|0|0|
```
...ersetzt werden durch:
```
hidden|termofuse_accepted|1
```
